# node-tool
一款多功能sspanel后端工具箱

### 使用方法
```shell
wget https://raw.githubusercontent.com/linux-terminal/node-tool/master/node.sh && bash node.sh
```

### 功能说明
- 一键安装docker版ssr
- 一键安装docker版偏移端口ssr
- 一键安装v2ray免费版
- 一键安装brook中转
- 一键安装BBR加速
- 一键重装系统
- 一键配置DNS解锁
- 一键设置swap
